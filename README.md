# Rose & Crown Reservation System

## Description
A full-stack web application that allows users to make reservations. Admin can log in and view all reservations.

## Tech Stack
- PHP
- MySQL
- XAMPP
- HTML/CSS

## Features
- User login system
- Add reservations
- View reservations (dashboard)
- Database integration

## Installation
1. Install XAMPP
2. Place project in htdocs
3. Start Apache and MySQL
4. Import database.sql in phpMyAdmin
5. Open http://localhost/rose_and_crown

## Environment Variables
- Database runs locally (localhost)
- Username: root
- Password: (empty)
